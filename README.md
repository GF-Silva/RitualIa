para rodar o backend:
abra a pasta;
crie um venv usando 'uv venv'
ative o venv
instale as libs usando 'uv sync'
rode usando 'fastapi dev main.py'

para rodar o frontend:
abra a pasta
crie um venv usando 'uv venv'
ative o venv
instale as libs usando 'uv sync'
rode usando 'python app/main.py'

abra o frontend em: localhost:5000 
# é importante abrir em localhost para o embbed funcionar
para testar o backend vá em: localhost:8000/docs
